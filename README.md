## Hi there 👋

Systems/IT person by day, tinkerer by night. Below is a rundown of everything in this account, grouped by category since the six pinned repos don't tell the whole story.

### 🏢 Datacenter / Infrastructure Automation
Tools for managing rack PDUs at scale.

- **[python-scripts-for-raritan-pdu-deployment](https://github.com/wh0oo/python-scripts-for-raritan-pdu-deployment)** — Python scripts for automated mass deployment of Raritan cabinet PDUs in a datacenter environment (password rotation, firmware deployment, naming, RADIUS/SNMPv3 staging). Not affiliated with Raritan — just a heavy user of their API.
- **[raritan-pdu-json-rpc-sdk](https://github.com/wh0oo/raritan-pdu-json-rpc-sdk)** — SDK/helper layer for talking to Raritan PX3-series PDUs over their JSON-RPC API.

### 🎮 Minecraft / Fabric Mods
All Fabric mods except the scarpet scripts.

- **[deepchat](https://github.com/wh0oo/deepchat)** — Fabric mod that integrates DeepSeek into Minecraft.
- **[brag](https://github.com/wh0oo/brag)** — Adds a `/brag` command for players (MC 26.1.2/26.2).
- **[ObsiCry](https://github.com/wh0oo/ObsiCry)** — Lightweight Fabric mod letting you build Nether portals from any mix of obsidian and crying obsidian.
- **[votebridge-mod](https://github.com/wh0oo/votebridge-mod)** — Bridges vote events from NuVotifier/VoteListener into Minecraft chat with placeholder and color support.
- **[suedew](https://github.com/wh0oo/suedew)** — `sudo`-style Fabric mod for server admins to simulate command execution as another player.
- **[Meteorolocracy](https://github.com/wh0oo/Meteorolocracy)** — Fabric mod for MC 1.21.7–1.21.9.
- **[cheerios](https://github.com/wh0oo/cheerios)** — Fabric mod.
- **[fortune](https://github.com/wh0oo/fortune)** — Fabric mod. "You can learn much from people who are different from you."
- **[sweepy](https://github.com/wh0oo/sweepy)** — Fabric mod.
- **[scarpet-scripts](https://github.com/wh0oo/scarpet-scripts)** — Scarpet scripts for running a Minecraft server.
